# Spring MCP Server

## Overview

This repository contains the source code for a Spring-based MCP server.

## Features

*   Built using Spring framework
*   Contains all CRUD tools

## Getting Started

### Prerequisites

*   Java 17 or higher
*   Spring Boot 3.5 or higher
*   Mongo db connection

### Build and Run

1.  Clone the repository: `git clone https://github.com/RajChauhan21/spring-mcp-server.git`
2.  Build the project: `mvn clean package`
3.  Run the server: `java -jar target/spring-mcp-server.jar`
